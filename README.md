基于C语言的libyaml库进行封装完善工作。主要是进行接口函数的编写。

src目录: 主要存放接口代码。

​	- source目录：接口代码的源文件目录。

​	- include目录：接口代码的头文件目录

​		- debug目录：调试常用的系统及头文件目录方便调用所设此目录。

target目录：GCC编译的过程文件以及可执行文件的存放目录。

yamlfile目录：用来存放yaml文件的目录。
